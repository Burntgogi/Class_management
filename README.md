# 학급 경영 자동화 시스템

교실에서 바로 사용할 수 있는 학급 운영 도구와 수업 보조 활동을 한곳에 모은 React 기반 대시보드입니다.

현재 포함된 핵심 기능:

- `청소구역 배정`: 학생 수와 구역 정원을 맞춰 공정하게 랜덤 배정하고, 결과 공개와 인쇄까지 처리합니다.
- `RBF Word Search`: 수업 도입, 전환 활동, 자투리 시간용 낱말 찾기 도구를 내부 페이지에서 바로 실행합니다.

## 리뉴얼 포인트

- 홈 화면을 교사용 대시보드 형태로 재구성했습니다.
- 카드 데이터를 분리해 도구 추가와 유지보수가 쉬워졌습니다.
- 청소구역 배정 화면의 정보 위계와 시각 스타일을 현대적으로 정리했습니다.
- 설정, 결과, 인쇄 흐름의 한국어 문구와 UI 톤을 정비했습니다.

## 기술 스택

- React 19
- TypeScript
- Vite
- Tailwind CSS
- DaisyUI
- React Router

## 실행 방법

```bash
npm install
npm run dev
```

배포용 빌드:

```bash
npm run build
```

## 프로젝트 구조

```text
components/   공통 UI 컴포넌트
data/         홈 대시보드용 카드 데이터
hooks/        배정 로직, 파일 저장/불러오기
pages/        홈, 청소구역 배정, 워드서치 페이지
types.ts      공통 타입 정의
```
